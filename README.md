## 🧩 Environment Variables

Rename `sample.env` to `.env` file and fill **only these** 👇

```env
API_ID=
API_HASH=
BOT_TOKEN=
MONGO_URL=
OWNER_USERNAME=
OWNER_ID=
BOT_NAME=
BOT_USERNAME=
UPDATES_CHANNEL=
```

### 📌 How to get them

- **API_ID & API_HASH** → https://my.telegram.org
- **BOT_TOKEN** → @BotFather
- **MONGO_URL** → MongoDB Atlas
- **OWNER_USERNAME** → Your username (without @)
- **BOT_USERNAME** → Bot username (without @)

## 🖥 VPS Deployment

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip git -y
git clone https://github.com/Aadishranjan/chatfight
cd chatfight
pip3 install -r requirements.txt
python3 run.py
```
